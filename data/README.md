# Data Directory

Place your dataset CSV file here.

## Requirements:
- Minimum 12 features
- Minimum 500 instances  
- CSV format
- Binary or multi-class classification

## Example datasets you can use:
1. **Heart Disease Dataset** (UCI)
2. **Wine Quality Dataset** (UCI)
3. **Bank Marketing Dataset** (UCI)
4. **Adult Income Dataset** (UCI)
5. Any Kaggle classification dataset meeting the requirements

## To generate sample datasets for testing:
```bash
cd model
python generate_sample_data.py
```

This will create sample datasets in this directory that you can use for initial testing.

**Note:** For the actual assignment submission, please use a real dataset from Kaggle or UCI repository.
